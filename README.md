 CRM Application (MERN Stack)

A simple Customer Relationship Management system built for my assignment.

 Features
* User Auth: Register and Login with secure passwords.
* CRM CRUD: Add, View, and Delete customer data.
* Responsive UI: Built with React and Tailwind CSS.
* Cloud Database: Connected to MongoDB Atlas.

How to Run
1. Server: Go to `/server`, run `npm install`, then `npm run dev`.
2. Client: Go to `/client`, run `npm install`, then `npm run dev`.
